## 给 AI 工具的提示词

```text
请先读取 https://docs.crazyrouter.com/llms.txt 判断相关页面。
如果需要完整上下文，再读取 https://docs.crazyrouter.com/llms-full.txt。
模型名称、价格和计费方式必须以 https://crazyrouter.com/pricing 为准。
如需确认当前 API Key 实际可用模型，请调用 GET https://api.crazyrouter.com/v1/models。
接入 OpenAI 兼容客户端时 base_url 使用 https://api.crazyrouter.com/v1。
接入 Claude Code / Anthropic 原生客户端时使用根域名 https://api.crazyrouter.com，不要追加 /v1。
东亚地区优先使用 https://cn.crazyrouter.com；OpenAI 兼容客户端对应 https://cn.crazyrouter.com/v1。
```

## 口径要求

- `llms.txt` 不是全量模型清单。
- 模型、价格和计费方式以 [Pricing 页面](https://crazyrouter.com/pricing) 为准。
- 当前 Token 能调用哪些模型，以 `GET https://api.crazyrouter.com/v1/models` 的返回结果为准。
- 东亚地区优先使用 `https://cn.crazyrouter.com`，OpenAI 兼容客户端使用 `https://cn.crazyrouter.com/v1`。
- 生成调用代码时，优先使用具体模型文档中已经验证过的端点示例。
