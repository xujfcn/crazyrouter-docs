# Kling Advanced Features

## Virtual Try-On

```
POST /kling/v1/images/kolors-virtual-try-on
```

Apply clothing images onto person photos.

### Request Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `model_name` | string | Yes | Model name |
| `human_image` | string | Yes | Person photo URL |
| `cloth_image` | string | Yes | Clothing image URL |
| `callback_url` | string | No | Callback URL |

### Request Example

```bash cURL
curl -X POST https://crazyrouter.com/kling/v1/images/kolors-virtual-try-on \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model_name": "kolors-virtual-try-on-v1",
    "human_image": "https://example.com/person.jpg",
    "cloth_image": "https://example.com/dress.jpg"
  }'
```

## Query Try-On Task

```
GET /kling/v1/images/kolors-virtual-try-on/{task_id}
```

```bash cURL
curl https://crazyrouter.com/kling/v1/images/kolors-virtual-try-on/kling_tryon_abc123 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

<Note>
  The only currently exposed Kling advanced capability is virtual try-on. Lip sync, video extension, and effects are not publicly opened yet and should not be treated as available endpoints.
</Note>
