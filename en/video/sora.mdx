# Sora (Temporarily Hidden)

Sora public examples are no longer shown as a recommended Crazyrouter integration path. To prevent users from copying examples for delisted or non-public models, this page has been removed from navigation and no longer includes request samples.

For new video integrations, use:

- [Veo 3.1](/en/video/veo): `veo-3.1-fast` or `veo-3.1-quality`
- [Unified Video API](/en/video/unified): `POST /v1/video/create`
- [OpenAI-compatible Video API](/en/video/openai-video): `POST /v1/videos`

<Note>
  If Sora returns to the public model list later, this page should be updated with the current model IDs, request examples, and validation notes.
</Note>
