# Sora

OpenAI Sora supports multiple API formats: Video Generations API, Unified Video API, OpenAI Video format, and native format.

## Model Selection

| Model | Description | Speed | Quality | Recommended Use |
|-------|-------------|-------|---------|-----------------|
| `sora-2` | Second generation standard | Medium | High | Production, daily use |
| `sora-2-pro` | Second generation professional | Slower | Highest | Professional work, fine control |

---

## Via Video Generations API (Recommended)

```
POST /v1/video/generations
```

This is the latest API format with more detailed task status and metadata.

```bash cURL
curl -X POST https://crazyrouter.com/v1/video/generations \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "sora-2",
    "prompt": "A serene Japanese garden in spring, with cherry blossoms gently falling",
    "size": "1280x720",
    "duration": 5
  }'
```

### Response Example

```json
{
  "id": "video_69b3a77020e88193b4476bb2be76b39400a8d08edef728f9",
  "object": "video",
  "model": "sora-2",
  "status": "queued",
  "progress": 0,
  "created_at": 1773381488,
  "seconds": "8",
  "size": "1280x720"
}
```

### Task Status

| Status | Description |
|--------|-------------|
| `queued` | Task submitted, waiting for processing |
| `in_progress` | Generating |
| `completed` | Generation completed |
| `failed` | Generation failed |

### Important Fields

- `generation_time`: Actual generation time (seconds)
- `expires_at`: Video URL expiration timestamp (typically 1 hour after generation)
- `thumbnail_url`: Video thumbnail
- `spritesheet_url`: Video spritesheet (for preview)

<Note>
  Video URLs expire after 1 hour. Please download and save promptly.
</Note>

## Storyboard Support

Define multi-scene videos via storyboard:

```bash cURL
curl -X POST https://crazyrouter.com/v1/videos \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "sora-2",
    "storyboard": [
      {
        "prompt": "A sunrise over the mountains",
        "duration": 5
      },
      {
        "prompt": "Camera pans down to reveal a village",
        "duration": 5
      },
      {
        "prompt": "Close-up of a farmer starting their day",
        "duration": 5
      }
    ],
    "size": "1920x1080"
  }'
```

## Performance Reference

Based on actual testing with `sora-2` model:

- **Queue wait time**: ~60-90 seconds
- **Generation time**: ~120-150 seconds
- **Total time**: ~3-4 minutes
- **Video size**: ~8 MB (8 seconds, 1280x720)

<Note>
  Generation time varies based on system load and video complexity. sora-2-pro mode takes longer but delivers higher quality. Storyboard mode is ideal for multi-scene narrative videos.
</Note>
