# Gemini Image Understanding

```
POST /v1beta/models/{model}:generateContent
```

As of March 23, 2026, Crazyrouter production has verified that `gemini-3-pro-preview` can read an image through `inlineData` and return a text description.

---

## Verified Minimal Request

```bash
curl "https://crazyrouter.com/v1beta/models/gemini-3-pro-preview:generateContent?key=YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "contents": [
      {
        "role": "user",
        "parts": [
          {"text": "What color is this image?"},
          {
            "inlineData": {
              "mimeType": "image/png",
              "data": "iVBORw0KGgoAAA..."
            }
          }
        ]
      }
    ]
  }'
```

Observed production response shape:

```json
{
  "candidates": [
    {
      "content": {
        "parts": [
          {
            "text": "Based on the image provided, the color is red."
          }
        ]
      }
    }
  ]
}
```
