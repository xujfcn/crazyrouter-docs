# Special Models (Temporarily Hidden)

The historical special-models page is no longer shown as a public API example page. For new integrations, use:

- [Latest Model Examples](/en/latest-model-examples)
- [List Models](/en/chat/openai/models)
- [Chat Completions](/en/chat/openai/completions)

<Note>
  Before production rollout, call `GET /v1/models` to confirm which model IDs are visible to your token. Models that are not in the public index or not visible to the current token should not be used in defaults or examples.
</Note>
