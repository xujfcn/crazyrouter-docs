Crazyrouter is an AI model API aggregation gateway that provides unified access to 300+ AI models. It supports major AI services including OpenAI, Anthropic Claude, Google Gemini, Midjourney, Suno, Kling, and more.

## Core Features

- **Unified Interface**: Access all models with a single API Key
- **Multi-Format Support**: Compatible with OpenAI, Anthropic, and Gemini native formats
- **300+ Models**: Covering chat, image, video, audio, music, and more
- **Pay-As-You-Go**: Flexible token-based billing
- **Smart Routing**: Automatic optimal channel selection with failover

## Supported API Types

| Type | Description | Example Models |
|------|-------------|----------------|
| Chat Completions | Text conversation, vision, reasoning | GPT-4o, Claude Opus, Gemini Pro |
| Image Generation | Text-to-image, image editing | DALL-E 3, Midjourney, Flux, Ideogram |
| Video Generation | Text-to-video, image-to-video | Sora, Kling, Luma, Veo 3.1, Runway |
| Audio Processing | TTS, STT | GPT-4o-mini-tts, Whisper |
| Music Generation | Song creation | Suno |
| Embeddings | Text vectorization | text-embedding-3-large |
| Reranking | Search result reranking | BAAI/bge-reranker-v2-m3 |
