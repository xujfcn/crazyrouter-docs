# GPT Realtime

GPT Realtime is not currently shown as a public Crazyrouter example. To avoid copyable examples for delisted or non-public models, this page has been removed from navigation and no longer includes WebSocket request samples.

For public audio workflows, use:

- [Speech-to-Text](/en/audio/stt): `whisper-1`
- [Text-to-Speech](/en/audio/tts): `tts-1`

<Warning>
  Realtime availability should be checked against `/api/pricing`, the dashboard model picker, and production API responses before publishing examples.
</Warning>
