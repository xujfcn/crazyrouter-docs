# GPT-4o Audio

GPT-4o Audio is not currently shown as a public Crazyrouter example. To avoid copyable examples for delisted or non-public models, this page has been removed from navigation and no longer includes request samples.

For public audio workflows, use:

- [Speech-to-Text](/en/audio/stt): `whisper-1`
- [Text-to-Speech](/en/audio/tts): `tts-1`

<Note>
  For audio understanding, check the current public model list, `GET /v1/models`, and the dashboard model picker before publishing examples.
</Note>
