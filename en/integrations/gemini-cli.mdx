## Overview

Gemini CLI is Google's command-line AI assistant. By configuring Crazyrouter's Gemini-native endpoint, you can use the latest Gemini model already verified in production directly from the terminal.

## Configuration

### Environment Variables

```bash
export GEMINI_API_KEY=sk-xxx
export GEMINI_BASE_URL=https://crazyrouter.com
```

### Configuration File

Edit `~/.gemini/settings.json`:

```json
{
  "apiKey": "sk-xxx",
  "baseUrl": "https://crazyrouter.com",
  "model": "gemini-3-pro-preview"
}
```

### Launch

```bash
gemini
```

## Recommended Model

| Model | Description |
|-------|-------------|
| `gemini-3-pro-preview` | Verified successfully in Crazyrouter production on March 23, 2026, and suited for the main Gemini CLI baseline |

<Note>
  Crazyrouter supports both the Gemini native API format and the OpenAI-compatible format. Gemini CLI uses the Gemini-native format, and the `gemini-3-pro-preview` example above has been validated in production.
</Note>
