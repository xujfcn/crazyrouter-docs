## Overview

ChatGPT Sidebar is a browser sidebar AI assistant extension that supports quick AI access on any web page.

## Configuration Steps

<Steps>
  <Step title="Install Extension">
    Install ChatGPT Sidebar from the Chrome Web Store.
  </Step>
  <Step title="Open Settings">
    Click the extension icon and go to `Settings`.
  </Step>
  <Step title="Configure API">
    - **API Provider**: Select `Custom API` or `OpenAI`
    - **API URL**: `https://crazyrouter.com/v1`
    - **API Key**: `sk-xxx`
    - **Model**: `gpt-4o`
  </Step>
  <Step title="Save">
    Save settings. Click the extension icon on any web page to start using it.
  </Step>
</Steps>

## Use Cases

- Select web text, right-click to translate/summarize with AI
- Chat directly with AI in the sidebar
- Ask questions about the current web page content

<Note>
  Different versions of the ChatGPT Sidebar extension may have slightly different configuration interfaces. The core settings are always the API URL and API Key.
</Note>
