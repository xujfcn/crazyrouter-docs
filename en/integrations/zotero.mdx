## Overview

Zotero GPT is an AI plugin for the Zotero reference manager, supporting paper summarization, translation, Q&A, and more.

## Configuration Steps

<Steps>
  <Step title="Install Plugin">
    Install the `zotero-gpt` or `Aria` AI plugin in Zotero.
  </Step>
  <Step title="Open Plugin Settings">
    Go to `Edit` → `Preferences` → `Zotero GPT` (or the corresponding plugin name)
  </Step>
  <Step title="Configure API">
    - **API URL**: `https://crazyrouter.com/v1`
    - **API Key**: `sk-xxx`
    - **Model**: `gpt-4o` (recommended for academic use)
  </Step>
  <Step title="Test">
    Select a reference, right-click and use the AI feature to test.
  </Step>
</Steps>

## Recommended Models

| Scenario | Recommended Model |
|----------|-------------------|
| Paper summarization | `gpt-4o` |
| Literature translation | `gpt-4o-mini` (cost-effective) |
| Deep analysis | `claude-opus-4-20250514` |

<Note>
  Different Zotero AI plugins may have slightly different configuration interfaces. The core settings are always the API URL and API Key.
</Note>
