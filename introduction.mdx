<Note>
  API 接入路径：OpenAI 兼容客户端通常使用 `https://crazyrouter.com/v1` 或 `https://cn.crazyrouter.com/v1`；Claude Code / Anthropic 原生客户端使用根域名，不要追加 `/v1`。详见 [API 端点](/api-endpoint)。
</Note>

Crazyrouter 是统一的 AI 模型 API 网关。你可以用一个 API Key 调用文本、图像、视频、音频、Embedding、Rerank 和第三方工具集成所需的模型能力。
