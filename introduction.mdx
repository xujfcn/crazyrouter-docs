Crazyrouter 是一个 AI 模型 API 聚合网关，提供对 300+ AI 模型的统一访问接口。支持 OpenAI、Anthropic Claude、Google Gemini、Midjourney、Suno、Kling 等主流 AI 服务。

## 核心特性

- **统一接口**：一个 API Key 访问所有模型
- **多格式支持**：兼容 OpenAI、Anthropic、Gemini 原生格式
- **300+ 模型**：覆盖聊天、图像、视频、音频、音乐等
- **按量计费**：灵活的 Token 计费方式
- **智能路由**：自动选择最优渠道，故障自动切换

## 支持的 API 类型

| 类型 | 说明 | 示例模型 |
|------|------|---------|
| 聊天补全 | 文本对话、识图、推理 | GPT-4o, Claude Opus, Gemini Pro |
| 图像生成 | 文生图、图编辑 | DALL-E 3, Midjourney, Flux, Ideogram |
| 视频生成 | 文生视频、图生视频 | Sora, Kling, Luma, Veo 3.1, Runway |
| 音频处理 | TTS、STT | GPT-4o-mini-tts, Whisper |
| 音乐生成 | 歌曲创作 | Suno |
| 嵌入向量 | 文本向量化 | text-embedding-3-large |
| 重排序 | 搜索结果重排 | BAAI/bge-reranker-v2-m3 |
