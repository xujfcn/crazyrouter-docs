# Gemini 图片生成

```
POST /v1/images/generations
```

Crazyrouter 对 Gemini 图片模型统一提供 OpenAI Images 兼容入口。用户侧不需要直接调用 Gemini 原生 `generateContent`，也不需要传 Gemini 原生 `generationConfig.imageConfig`。

| 推荐公开模型名 | 兼容底层模型名 |
|------|------|
| `nano-banana` | `gemini-2.5-flash-image` |
| `nano-banana-pro` | `gemini-3-pro-image-preview` |
| `nano-banana-2` | `gemini-3.1-flash-image-preview` |

<Warning>
  Gemini 图片生成属于长耗时请求。生产接入请优先使用主承接线路 `https://cn.crazyrouter.com/v1`，完整生成端点为 `https://cn.crazyrouter.com/v1/images/generations`。
</Warning>

<Note>
  推荐在客户代码里使用 `nano-banana`、`nano-banana-pro`、`nano-banana-2`。如果你的系统已经直接写了 `gemini-2.5-flash-image`、`gemini-3-pro-image-preview`、`gemini-3.1-flash-image-preview`，也可以继续通过同一个 Images API 入口调用。
</Note>

## 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `model` | string | 是 | `nano-banana`、`nano-banana-pro`、`nano-banana-2`，或对应 Gemini canonical 名 |
| `prompt` | string | 是 | 生成或编辑指令 |
| `image_input` | string 或 string[] | 否 | 参考图 URL；不传则为文生图 |
| `resolution` | string | 否 | `1K`、`2K`、`4K` |
| `aspect_ratio` | string | 否 | 如 `1:1`、`16:9`、`9:16`、`21:9`、`auto` |
| `output_format` | string | 否 | `png`、`jpeg` / `jpg` |
| `output_compression` | integer | 否 | `0-100`，用于 JPEG 压缩质量 |
| `n` | integer | 否 | 生成数量，默认 `1` |

不要传这些上游私有字段：`generationConfig`、`imageConfig`、`imageOutputOptions`、`image_output_options`。

## 请求示例

```bash
curl -X POST https://cn.crazyrouter.com/v1/images/generations \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "nano-banana-2",
    "prompt": "A simple red square on a white background, clean vector style",
    "resolution": "1K",
    "aspect_ratio": "1:1",
    "output_format": "png"
  }'
```

响应保持 OpenAI Images 风格：

```json
{
  "created": 1778990000,
  "data": [
    {
      "url": "https://media.crazyrouter.com/task-artifacts/2026/05/17/sync-image/request-id-1.png"
    }
  ]
}
```

## 参考图编辑

```bash
curl -X POST https://cn.crazyrouter.com/v1/images/generations \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "nano-banana-pro",
    "prompt": "把参考图改成高级电商海报，保留主体，背景更简洁",
    "image_input": [
      "https://example.com/reference.png"
    ],
    "resolution": "1K",
    "aspect_ratio": "1:1",
    "output_format": "jpeg",
    "output_compression": 85
  }'
```

<Note>
  如果你确实需要 Gemini 原生 API 形态，请参考 [Gemini 原生格式](/chat/gemini/native)。但对于图片生成和编辑，推荐客户侧统一使用本页的 Images API。
</Note>
