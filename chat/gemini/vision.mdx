# Gemini 图片理解

```
POST /v1beta/models/{model}:generateContent
```

截至 2026 年 3 月 23 日，Crazyrouter 生产环境已验证 `gemini-3-pro-preview` 可以通过 `inlineData` 读取图片并返回文本描述。

---

## 已验证的最小请求

```bash
curl "https://crazyrouter.com/v1beta/models/gemini-3-pro-preview:generateContent?key=YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "contents": [
      {
        "role": "user",
        "parts": [
          {"text": "What color is this image?"},
          {
            "inlineData": {
              "mimeType": "image/png",
              "data": "iVBORw0KGgoAAA..."
            }
          }
        ]
      }
    ]
  }'
```

生产环境实际返回的关键形态：

```json
{
  "candidates": [
    {
      "content": {
        "parts": [
          {
            "text": "Based on the image provided, the color is red."
          }
        ]
      }
    }
  ]
}
```

<Note>
  当前页面只保留了图片输入这条已复测成功的能力。视频、音频与更大文件场景，本轮没有逐项复测，因此不再继续写成当前主文档。
</Note>
