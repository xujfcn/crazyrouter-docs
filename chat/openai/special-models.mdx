## 当前建议

- 如果你要判断某个特殊模型今天能不能用，先查 [GET /v1/models](/chat/openai/models)
- 模型出现在列表里，也不代表当天就一定能成功调用
- 对于这类模型，首次联调建议先做一次最小探针请求，再写入正式流程

## 最小探针模板

```bash cURL
curl https://crazyrouter.com/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "YOUR_SPECIAL_MODEL",
    "messages": [
      {
        "role": "user",
        "content": "Hello"
      }
    ],
    "max_tokens": 64
  }'
```

如果返回：

- 正常 `choices`
  说明当前通道可用
- `model_not_found`
  说明当前模型不在可用范围内
- `temporarily unavailable` / `get_channel_failed`
  说明模型名可见，但当前通道不可用

<Warning>
  特殊模型页最容易过时。对于翻译、OCR、实验性专用模型，建议始终以当天的 `/v1/models` 结果和一次真实请求为准。
</Warning>
