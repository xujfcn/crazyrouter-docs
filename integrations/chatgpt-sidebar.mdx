## 概述

ChatGPT Sidebar 是一款浏览器侧边栏 AI 助手扩展，支持在任意网页上快速调用 AI。

## 配置步骤

<Steps>
  <Step title="安装扩展">
    从 Chrome 网上应用店安装 ChatGPT Sidebar。
  </Step>
  <Step title="打开设置">
    点击扩展图标，进入 `Settings`。
  </Step>
  <Step title="配置 API">
    - **API Provider**: 选择 `Custom API` 或 `OpenAI`
    - **API URL**: `https://crazyrouter.com/v1`
    - **API Key**: `sk-xxx`
    - **Model**: `gpt-4o`
  </Step>
  <Step title="保存">
    保存设置后，在任意网页上点击扩展图标即可使用。
  </Step>
</Steps>

## 使用场景

- 选中网页文本，右键使用 AI 翻译/总结
- 在侧边栏中直接与 AI 对话
- 对当前网页内容进行提问

<Note>
  不同版本的 ChatGPT Sidebar 扩展配置界面可能略有差异，核心配置项都是 API URL 和 API Key。
</Note>
