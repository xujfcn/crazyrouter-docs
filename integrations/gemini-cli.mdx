## 概述

Gemini CLI 是 Google 的命令行 AI 助手。通过配置 Crazyrouter 的 Gemini 原生入口，你可以在命令行中直接使用已在生产环境验证过的 Gemini 最新模型。

## 配置方法

### 环境变量

```bash
export GEMINI_API_KEY=sk-xxx
export GEMINI_BASE_URL=https://crazyrouter.com
```

### 配置文件

编辑 `~/.gemini/settings.json`：

```json
{
  "apiKey": "sk-xxx",
  "baseUrl": "https://crazyrouter.com",
  "model": "gemini-3-pro-preview"
}
```

### 启动

```bash
gemini
```

## 推荐模型

| 模型 | 说明 |
|------|------|
| `gemini-3-pro-preview` | 2026 年 3 月 23 日已在生产环境实测成功，适合作为 Gemini CLI 主基线 |

<Note>
  Crazyrouter 同时支持 Gemini 原生 API 格式和 OpenAI 兼容格式。Gemini CLI 使用 Gemini 原生格式，文档中的 `gemini-3-pro-preview` 已完成生产环境实测。
</Note>
