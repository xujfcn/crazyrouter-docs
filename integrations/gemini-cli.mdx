## 概述

Gemini CLI 是 Google 的命令行 AI 助手。通过配置，可以将其接入 Crazyrouter 使用 Gemini 系列模型。

## 配置方法

### 环境变量

```bash
export GEMINI_API_KEY=sk-xxx
export GEMINI_BASE_URL=https://crazyrouter.com
```

### 配置文件

编辑 `~/.gemini/settings.json`：

```json
{
  "apiKey": "sk-xxx",
  "baseUrl": "https://crazyrouter.com",
  "model": "gemini-2.5-pro"
}
```

### 启动

```bash
gemini
```

## 可用模型

| 模型 | 说明 |
|------|------|
| `gemini-2.5-pro` | 最强推理能力 |
| `gemini-2.5-flash` | 快速响应 |
| `gemini-2.0-flash` | 上一代快速模型 |

<Note>
  Crazyrouter 同时支持 Gemini 原生 API 格式和 OpenAI 兼容格式。Gemini CLI 使用原生格式。
</Note>
