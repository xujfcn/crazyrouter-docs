## 配置步骤

<Steps>
  <Step title="打开设置">
    打开 Cursor，点击左下角齿轮图标，选择 `Settings` → `Models`
  </Step>
  <Step title="配置 API">
    - **OpenAI API Key**: 输入你的 `sk-xxx` 密钥
    - **Override OpenAI Base URL**: 输入 `https://crazyrouter.com/v1`
  </Step>
  <Step title="选择模型">
    在模型列表中选择需要的模型，如 `gpt-4o`、`claude-sonnet-4-20250514` 等。也可以点击 `+ Add Model` 添加自定义模型名称。
  </Step>
  <Step title="验证连接">
    在编辑器中按 `Ctrl+K`（或 `Cmd+K`）打开 AI 面板，输入任意问题测试是否正常响应。
  </Step>
</Steps>

## 使用技巧

- **Tab 补全**：Cursor 的 Tab 自动补全会使用配置的模型
- **Ctrl+K**：内联编辑，选中代码后按 Ctrl+K 可以让 AI 修改
- **Ctrl+L**：打开侧边栏聊天
- **Composer**：按 `Ctrl+I` 打开 Composer 进行多文件编辑

<Warning>
  Cursor 的部分高级功能（如 Agent 模式）可能需要特定模型支持。建议使用 `gpt-4o` 或 `claude-sonnet-4-20250514` 以获得最佳体验。
</Warning>
