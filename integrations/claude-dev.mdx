## 概述

Claude Dev 是 VS Code 里的 AI 编程扩展，适合把 Claude 风格的代码解释、重构建议和多轮协作接到 Crazyrouter。公开文档里建议优先使用 OpenAI Compatible 路线，因为这条链路在本站文档体系里更统一，也更容易排障。

## 推荐配置

<Steps>
  <Step title="安装扩展">
    在 VS Code 扩展市场搜索 `Claude Dev` 并安装。
  </Step>
  <Step title="打开设置">
    点击侧边栏的 Claude Dev 图标，进入设置。
  </Step>
  <Step title="选择 Provider">
    优先选择 `OpenAI Compatible` 作为 API Provider。
  </Step>
  <Step title="填写配置">
    - **Base URL**: `https://crazyrouter.com/v1`
    - **API Key**: `sk-xxx`
    - **Model**: `claude-sonnet-4-6`
  </Step>
</Steps>

## 推荐模型

| 场景 | 模型 |
|------|------|
| 日常编程 | `claude-sonnet-4-6` |
| 更重推理与规划 | `claude-opus-4-6-thinking` |

<Note>
  `claude-sonnet-4-6` 与 `claude-opus-4-6-thinking` 都属于当前已验证过的 Claude 最新口径。若只是先做 VS Code 内的最小验证，先用 `claude-sonnet-4-6`。
</Note>
