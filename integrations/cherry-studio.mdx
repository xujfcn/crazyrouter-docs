## 概述

Cherry Studio 是一款功能丰富的 AI 桌面客户端，支持多模型对话、MCP 工具、推理模型等。

## 基础配置

<Steps>
  <Step title="打开设置">
    启动 Cherry Studio，点击左下角 `设置` → `模型服务`
  </Step>
  <Step title="添加服务商">
    点击 `添加服务商`，选择 `OpenAI 兼容` 类型
  </Step>
  <Step title="填写信息">
    - **名称**: `Crazyrouter`
    - **API 地址**: `https://crazyrouter.com/v1`
    - **API Key**: `sk-xxx`
  </Step>
  <Step title="添加模型">
    点击 `获取模型列表`，系统会自动拉取可用模型。也可以手动添加模型名称。
  </Step>
</Steps>

## MCP 工具配置

Cherry Studio 支持 MCP（Model Context Protocol）工具：

1. 进入 `设置` → `MCP 服务`
2. 添加 MCP 服务器配置
3. 在对话中启用 MCP 工具

## 推理模型配置

对于支持推理的模型（如 `o1`、`o3`），Cherry Studio 会自动识别并显示思考过程：

1. 添加模型时选择 `o1`、`o3-mini` 等推理模型
2. 对话时会自动展示推理链

## 推荐设置

| 设置项 | 推荐值 |
|--------|--------|
| 温度 | 0.7（创意任务）/ 0（精确任务） |
| 最大 Token | 4096 |
| 流式输出 | 开启 |

<Note>
  Cherry Studio 支持同时配置多个服务商，可以在对话中随时切换不同的模型。
</Note>
