Crazyrouter - это API-шлюз для AI-моделей, который дает единый доступ к 300+ моделям. Он поддерживает основные AI-сервисы, включая OpenAI, Anthropic Claude, Google Gemini, Midjourney, Suno, Kling и другие.

## Основные возможности

- **Единый интерфейс**: доступ ко всем моделям через один API-ключ
- **Поддержка разных форматов**: совместимость с OpenAI, Anthropic и нативным форматом Gemini
- **300+ моделей**: чат, изображения, видео, аудио, музыка, embeddings и rerank
- **Оплата по факту использования**: гибкая тарификация на основе токенов
- **Умная маршрутизация**: автоматический выбор оптимального канала и failover

## Поддерживаемые типы API

| Тип | Описание | Примеры моделей |
|-----|----------|-----------------|
| Chat Completions | Текстовый диалог, зрение, reasoning | GPT-4o, Claude Opus, Gemini Pro |
| Генерация изображений | Text-to-image, редактирование изображений | DALL-E 3, Midjourney, Flux, Ideogram |
| Генерация видео | Text-to-video, image-to-video | Sora, Kling, Luma, Veo 3.1, Runway |
| Обработка аудио | TTS, STT | GPT-4o-mini-tts, Whisper |
| Генерация музыки | Создание песен | Suno |
| Embeddings | Векторизация текста | text-embedding-3-large |
| Rerank | Переранжирование результатов поиска | BAAI/bge-reranker-v2-m3 |
