<Note>
  API routes: OpenAI-compatible клиенты обычно используют `https://crazyrouter.com/v1` или `https://cn.crazyrouter.com/v1`; Claude Code и Anthropic-native клиенты используют корневой домен без `/v1`. См. [API Endpoints](/ru/api-endpoint).
</Note>

Crazyrouter - это единый API-шлюз для AI-моделей. Один API-ключ дает доступ к текстовым, image, video, audio, embedding, rerank и tool-integration сценариям.
