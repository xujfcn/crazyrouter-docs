# Flux

<Warning>
  The current Crazyrouter pricing page does not expose a directly callable Flux `model_name`. This page no longer provides Flux request examples, and old `flux-*` model IDs should not be used in production configs.
</Warning>

## Current Alternatives

For image generation or editing, prefer models that are currently exposed on the pricing page:

| Use case | Suggested models |
|----------|------------------|
| OpenAI Images | `gpt-image-2`, `dall-e-3` |
| Gemini Images | `nano-banana`, `nano-banana-2`, `nano-banana-pro` |
| Grok Images | `grok-4-image` |
| Qwen Images | `qwen-image-2.0`, `qwen-image-plus`, `qwen-image-max` |
| Doubao Images | `doubao-seedream-4-5`, `doubao-seedream-5-0` |

## How To Confirm

Use `https://crazyrouter.com/pricing` and the `model_name` values returned by `GET /api/pricing` as the source of truth. Only models shown there should be used as documentation examples or customer-facing defaults.
