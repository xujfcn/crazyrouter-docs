# Flux

<Warning>
  当前本站 pricing 页面没有公开可直接调用的 Flux `model_name`。因此本文不再提供 Flux 请求示例，也不建议在生产配置里填写旧的 `flux-*` 模型名。
</Warning>

## 当前可用替代

如果你需要图像生成或图像编辑，请优先使用 pricing 页面当前公开的模型：

| 场景 | 建议模型 |
|------|----------|
| OpenAI 图片接口 | `gpt-image-2`、`dall-e-3` |
| Gemini 图片接口 | `nano-banana`、`nano-banana-2`、`nano-banana-pro` |
| Grok 图片接口 | `grok-4-image` |
| Qwen 图片接口 | `qwen-image-2.0`、`qwen-image-plus`、`qwen-image-max` |
| Doubao 图片接口 | `doubao-seedream-4-5`、`doubao-seedream-5-0` |

## 如何确认

以 `https://crazyrouter.com/pricing` 和 `GET /v1/models` 返回的当前 Token 可见模型为准。只有公开可见且可提交成功的模型名才适合作为文档示例和客户配置默认值。
