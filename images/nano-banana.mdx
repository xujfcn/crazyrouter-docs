# Nano Banana

```
当前仅用于受控测试
POST /v1beta/models/gemini-2.5-flash-image:generateContent
```

截至 `2026-04-14`，Crazyrouter 对 `nano-banana` 的最新生产复测结论如下：

- `nano-banana` 在服务端当前映射到 `gemini-2.5-flash-image`
- 不再推荐通过 `POST /v1/images/generations` 调用 `nano-banana`
- `gemini-2.5-flash-image:generateContent` 当前可以返回图片结果
- 但这条原生链路的计费一致性仍在复核中，因此当前只适合受控测试，不应写成稳定生产接入方案

<Warning>
  `nano-banana` 当前不要再作为 `/v1/images/generations` 的推荐模型写给客户。即使 Gemini 原生链路能成功返回结果，也先不要把它写成稳定生产能力，直到计费一致性复核完成。
</Warning>

## 当前状态

| 项目 | 当前状态 | 说明 |
|------|------|------|
| 别名映射 | 已确认 | `nano-banana -> gemini-2.5-flash-image` |
| Gemini 原生 `generateContent` | 可返回结果 | 当前仅建议受控测试 |
| 原生链路计费 | 有风险 | `2026-04-14` 生产复测里，请求成功与实际 `used_quota` 增量不一致 |
| OpenAI Images `POST /v1/images/generations` | 不推荐 | 生产与本地 `4000` 都复现错误 |

## 当前风险

### 1. 不再推荐 `/v1/images/generations`

`nano-banana` 走公开 Images 兼容路径时，`2026-04-14` 的生产与本地 `4000` 复测都命中了同类失败：

- `Unknown name "imageOutputOptions" at 'generation_config.image_config'`
- 请求失败后仍出现异常计费样本

这说明当前公开 Images 兼容链路并不适合作为 `nano-banana` 的客户文档入口。

### 2. 原生路径仍有计费一致性风险

当前生产环境中：

- `POST /v1beta/models/gemini-2.5-flash-image:generateContent` 可以返回图片
- 但同次探针里，消费日志中的 `quota` 与 token 侧 `used_quota` 增量不一致

因此这条路径当前只能写成“受控测试可用”，不能写成“稳定生产接入”。

## 当前建议

1. 如果你需要现在就稳定接图片生成或图片编辑，优先参考 [Nano Banana 2](/images/nano-banana-2)
2. 如果你必须评估 `nano-banana`，请只走 Gemini 原生 `generateContent`
3. 在计费一致性复核完成前，不要把 `nano-banana` 接入正式生产流量
4. 不要再使用 `POST /v1/images/generations` + `model: "nano-banana"`

## 受控测试模板

下面的请求模板只用于当日受控测试，不代表当前已经适合稳定生产接入：

```bash
curl "https://crazyrouter.com/v1beta/models/gemini-2.5-flash-image:generateContent?key=YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "contents": [
      {
        "role": "user",
        "parts": [
          {
            "text": "Generate an IMAGE of a clean banana icon on a white background. Return image output."
          }
        ]
      }
    ],
    "generationConfig": {
      "responseModalities": ["TEXT", "IMAGE"]
    }
  }'
```

## 旧路径说明

以下旧写法当前已不再推荐：

```bash
curl -X POST https://crazyrouter.com/v1/images/generations \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "nano-banana",
    "prompt": "..."
  }'
```

原因不是“文档风格调整”，而是这条链路当前在生产和本地复测里都不健康。

  如果你的目标是当前 Crazyrouter 上相对健康、可继续接入的 Gemini 图片路径，请看 [Gemini 图片生成](/chat/gemini/image-gen)；如果你的目标是当前更可用的 Nano Banana 系列入口，请优先看 [Nano Banana 2](/images/nano-banana-2)。
</Note>
